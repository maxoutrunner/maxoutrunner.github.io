TIC-80 CHEAT SHEET

    • Render
    • TIC()
      Called each frame at 60FPS
    • scanline(line)
      Called on every line render.
    • OVR()
      Called on each frame. Draws on separate layer.
    • clip(x, y, w, h)
      Limits render area to size W, H, starting at X, Y.
    • Drawing
    • cls([color])
      Fill screen with palette colour 0 or COLOR if specified.
    • print(text [, x, y][, c][, fixed=false][, scale=1])
      Print TEXT with system font at X, Y, of colour C, with FIXED character width, at SCALE. Returns text width.
    • font(text, x, y, key, w, h, fixed, scale)
      Prints TEXT at X, Y with custom font. KEY is transparency colour, W and H are character width and height used for spacing, fixed is a flag indicating wether to fix the width of the characters.
    • spr(id, x, y[, key=-1][, scale=1][, flip=0][, rotate=0][, w=1, h=1])
      Draw sprite ID at screen X, Y, with transparet colour of KEY, at SCALE, being FLIPPED: 1-horisontally, 2-vertically, 3-both; ROTATEd: 1-90*, 2-180*, 3-270*. If it's a composite sprite, it's W sprites wide, H sprites high.
    • pix(x, y, c)
      Put a pixel of colour C at X, Y on the screen.
    • line(x0, y0, x1, y1, c)
      Draw a 1px line of colour C from x0, y0, to x1, y1.
    • circ(x, y, r, c)
      Draw a filled with colour C circle with it's center at X, Y, it's radius R.
    • circb(x, y, r, c)
      Draw a ring of 1px of colour C with it's center at X, Y, it's radius R.
    • rect(x, y, w, h, c)
      Draw a filled rectangle of colour C, with the top left corner at X, Y, of width W and height H.
    • rectb(x, y, w, h, c)
      Draw a 1px rectangle of colour C, with the top left corner at X, Y, of width W and height H.
    • tri(x1, y1, x2, y2, x3, y3, c)
      Draw a filled triangle of colour C and the vertices at X1, Y1, X2...
    • textri(x1, y1, x2, y2, x3, y3, u1, v1, u2, v2, u3, v3[, use_map=false][, key=-1])
      Draw a triangle with vertices at X1, Y1, X2... , filled with texture.
    • Map
    • map(x, y, w, h, sx, sy, key, scale, remap)
      Draw a map segment from the mapspace, starting at tile X, Y, of width W and height H, starting at screen coordinates SX and SY, with KEY as transparent colour(-1 as none), at SCALE. Remap is a function that is called every tile render, expected to return the tile to be drawn instead of a current tile.
    • mset(x, y, id)
      Set map tile at X, Y, to the sprite ID>
    • mget(x, y)
      returns the sprite id at the given X and Y map coordinate
    • Controls
    • Player 1 buttons
      Up: 0, Down: 1, Left: 2, Right: 3, A: 4, B: 5
    • Player 2 buttons
      Up: 8, Down: 9, Left: 10, Right: 11, A: 12, B: 13
    • btn(id)
      Test if button ID was pressed.
    • btnp(id[, hold, period])
      TRUE only in the moment when button ID is pressed and, after the HOLD time is elapsed the function return TRUE every time PERIOD is passed.
    • key(code)
      True if key CODE on keyboard is pressed
    • keyp(code[, hold period])
      TRUE only in the moment when button CODE is pressed and, after the HOLD time is elapsed the function return TRUE every time PERIOD is passed.
    • mouse()
      Returns X, Y, PRESSED: the current coordinates of the mouse and the mouse button state.
    • Sound
    • music([track=-1][,frame=-1][,row=-1][,loop=true])
      Plays music track TRACK at frame FRAME, row ROW, (not) to be LOOPed. music() will stop the music
    • sfx(id [, note][, duration=-1][, channel=0][, volume=15][, speed=0])
      Play the sound ID, -1 to stop. Or play a custom note.
    • RAM
    • memcpy(toaddr, fromaddr, len)
      Copies RAM block of LEN bytes from FROMADDR to TOADDR.
    • memset(addr, val, len)
      Write a block of ram of LEN length at ADDR, with decimal value VAL.
    • peek(addr)
      Returns a byte from ADDR in decimal format.
    • peek4(addr)
      Returns 4 bits from ADDR in decimal
    • poke(addr, val)
      Write VAL to ram address ADDR
    • poke4(addr)
      Write 4 bits of VAL to ram address ADDR
    • System
    • exit()
      Interrupt program execution and return to the console
    • reset()
      Reset cartridge execution.
    • trace(msg[, c])
      Print MSG to terminal at colour C.
    • time()
      Returns number of seconds from the game start, in float.
    • pmem(index[, val])
      Return or save VAL(if specified) at the cartridges' persistent memory cell INDEX, ranging from 0 to 255
    • sync(mask=0, bank=0, toCart=false)
      Copy data from/to cart, set at TOCART, at memory bank BANK, with binary mask MASK. 0 to move all. Add 1 to move tiles, add 2 for sprites, 4 for map, 8 for sfx, 16 for music, 32 for palette.
